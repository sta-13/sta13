Resultados de la Búsqueda

Ingresa el título de una película y haz clic en "Buscar Película" para ver información detallada.

Instrucciones de Uso

  1. Ingresa el título de una película en inglés (ej: "The Dark Knight", "Pulp Fiction")
  2. Haz clic en "Buscar Película" para obtener información de la API de OMDB
  3. Si tienes tu propia API key de OMDB, ingrésala en el campo correspondiente para evitar límites de uso
  4. Haz clic en "Limpiar Búsqueda" para comenzar de nuevo
